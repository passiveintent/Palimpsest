# Fixture plots

PNGs here are generated by `src/pconformal/synth/plotting.py` for eyeballing
the synth generators' output during development. They are not golden
images — matplotlib rendering isn't guaranteed byte-identical across
environments — so they're gitignored (`*.png` in this directory) and
regenerated on demand rather than committed.

Regenerate all of them:

```sh
uv run python -c "from pconformal.synth.plotting import ALL_PLOTS; [p() for p in ALL_PLOTS]"
```

Or call any single `plot_*` function from `pconformal.synth.plotting`
directly for one generator.
