# Contributing

Palimpsest is currently source-available under the Business Source License 1.1
(`BUSL-1.1`).

Outside code contributions are not accepted unless they are covered by a signed
Contributor License Agreement (CLA). The agreement must grant the right to
relicense, commercially license, and otherwise distribute the contribution as
part of Palimpsest.

This repository uses [CLA Assistant](https://github.com/cla-assistant/cla-assistant)
to enforce this automatically: opening a pull request with code, tests, or
other copyrightable implementation material triggers a CLA-signing check, and
the PR cannot be merged until it passes. Issues, bug reports, design
discussion, and documentation feedback don't require a signed CLA.

For contribution and commercial licensing questions, contact
purushottam@passiveintent.dev.
