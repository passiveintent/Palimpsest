# ADR-010: Dimensional Slicing

**Status:** Accepted (pre-alpha; implementation pending)

## Problem

Ad-hoc PromQL over arbitrary dimensions after encode.

## Decision

Keyframes persisted as an exact, forever-queryable PromQL layer (Layer 1);
declared sketch-cube views for finer grains; group-lasso promoted from
backlog to prerequisite for correlated-failure recovery.
